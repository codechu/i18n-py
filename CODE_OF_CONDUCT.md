# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful, inclusive, and patient.
- Assume good faith; when giving criticism, focus on the work, not the person.
- Harassment, personal attacks, and discriminatory language are not accepted in any form.
- Repo maintainers have the authority to enforce these rules; persistent
  violations may result in a ban from the project's spaces.

## Reporting

Email `conduct@codechu.com` — reports are kept confidential.

The full Contributor Covenant text is the authoritative version; this
file is a quick reference.
